Esercizio 3

Realizza una pagina Angular come in figura.

I meme possono essere scelti a piacere, purché non siano cose oscene!

  1. Crea un’applicazione Angular chiamata esercizio-3
  2. Crea 3 componenti oltre all’app component
  3. Usa la stessa struttura dei componenti che vedi nel grafico ad albero
  4. Il second component deve contenere la tabella
  5. Usa la notazione {{ }} per recuperare il nome e la descrizione dei meme, contenute all’interno di variabili definite nel file component.ts.
    Le variabili devono essere contenute dentro un array.
  6. Il third component deve contenere il numero di like del primo meme, una barra e un bottone.
  1. Il numero deve essere inizializzato dal costruttore del componente e deve essere scelto a caso tra 0 e 100.
  2. La barra deve essere riempita in modo congruo al numero di like.
  3. Il pulsante deve aggiungere un like. Il numero massimo di like è 100.
  1. Il fourth component deve contenere un icona di caricamento di angular material.
  2. Le immagini devono essere adeguatamente ridimensionate usando il css.
  3. Usa i css per colorare l’immagine come nella figura
  4. Aggiungere il seguente comportamento al third component:
  1. Se il numero di like è minore di 33, la barra e il tasto devono essere colorati di blu (primary)
  2. Se il numero di like è maggiore o uguale a 33 e minore di 66, la barra e il tasto devono essere colorati di rosa (accent)
  3. Negli altri casi, la barra e il tasto devono essere colorati di rosso (warn)
