BOOTSTRAP 5

Modalità di lavoro.

Crea una cartella BOOTSTRAP dove salverai tutti i tuoi esercizi su Bootsrap

Per ogni documento crea una cartella chiamata BTS_X

Dove X + il numero del documento (in questo caso BTS_1).

Per ogni esercizio all’interno del documento crea un file chiamato come l’esercizio (ES: BTS_1.1.html)

Consegna gli esercizi su un repository git hub chiamato “Esercizi Bootstrap”

Usa come riferimento il sito Bootstrap 5 Tutorial (w3schools.com)

NOTA! Stiamo studiando la versione 5 di Bootstrap, dunque se cerchi altre risorse su web fai attenzione che si stia parlando della versione 5.

Puoi anche fare riferimento alla documentazione ufficiale di Bootstrap:

Introduction · Bootstrap v5.3 (getbootstrap.com)

In ogni esercizio rispondi alle domande provando le varie funzioni di Bootstrap.
