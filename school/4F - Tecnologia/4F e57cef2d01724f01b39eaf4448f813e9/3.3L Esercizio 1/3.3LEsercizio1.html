Esercizio 1

  1. Apri il repository con gitpod con l’url generato attraverso github es: http://gitpod.io#TUO_URL
  2. Esegui il push su github ogni volta che finisci di fare un punto (git add*, git commit -m “punto x”, git push origin master/main).  I punti vanno svolti in ordine
  3. Esegui il push solamente quando sei sicuro che il progetto realizzato funzioni.
  4. Progetti non funzionanti (es: che non compilano) riceveranno valutazione pari a 3.

Realizza una pagina Angular come in figura.

  1. Crea un’applicazione Angular chiamata tuo-cognome
  2. Crea 3 componenti oltre all’app component
  3. Dentro ciascun template scrivi quello che vedi in figura
  4. Usa la stessa struttura dei componenti che vedi nel grafico di destra
  5. Usa la notazione {{ }} per recuperare il testo delle parole colorate (app, secondo, terzo, quarto) all’interno di una variabile name definita nel file component.ts
  6. Usa i css per colorare l’immagine come nella figura di sinistra
