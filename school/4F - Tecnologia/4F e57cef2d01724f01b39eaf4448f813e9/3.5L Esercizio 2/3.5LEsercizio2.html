Esercizio 2 - Memes

Realizza una pagina Angular come in figura.

I meme possono essere scelti a piacere, purché non siano cose oscene!

  1. Crea un’applicazione Angular chiamata esercizio-2
  2. Crea 3 componenti oltre all’app component
  3. Usa la stessa struttura dei componenti che vedi nel grafico ad albero
  4. L’app component deve contenere la frase con le parole colorate.
  5. Usa la notazione {{ }} per recuperare il testo delle parole colorate all’interno di variabili name definita nel file component.ts.
  6. Il second component deve contenere la tabella
  7. Usa la notazione {{ }} per recuperare il nome e la descrizione dei meme, contenute all’interno di variabili definite nel file component.ts
  8. Il third component deve contenere l’immagine del primo meme; il fourth component deve contenere l’immagine del secondo meme.
  9. Le immagini devono essere adeguatamente ridimensionate usando il css.
  10. Usa i css per colorare l’immagine come nella figura
