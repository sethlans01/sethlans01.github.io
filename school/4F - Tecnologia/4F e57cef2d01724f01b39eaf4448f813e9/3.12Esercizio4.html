Esercizio 4

Realizza una web app Angular come segue.

  1. La web app deve avere 4 pagine, di cui una di benvenuto.
  2. Il tema della web app è il tuo cantante/band preferito.
  3. Le 3 pagine di contenuto possono contenere qualsiasi argomento che riguarda il cantante o la band.
  4. In ogni pagina ci devono essere almeno tre componenti
  5. Ogni pagina deve avere una barra di navigazione, che permette di raggiungere tutte le altre pagine
  6. La barra di navigazione deve essere realizzata come un componente di Angular
  7. Usa gli elementi o di Bootstrap o di Angular Material (è obbligatorio usare almeno una delle due alternative)
  8. In ogni pagina ci deve essere almeno una delle tre tipologie di data binding
  9. In generale, nella web app, tutti e tre i tipi di data binding devono essere usati almeno una volta
  10. Dare ai componenti una struttura ad albero sensata
  11. Il progetto deve essere committato e pushato su una repo di GitHub

NB: chiunque non si metta a fare il progetto prenderà una nota disciplinare e un voto in meno nella prossima verifica di Angular. (Alla fine di ogni lezione controllo quello che avete fatto).
