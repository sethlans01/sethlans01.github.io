Usare Bootstrap su Angular

Nel nostro progetto di Angular è possibile integrare anche Bootstrap, così da poter usare i suoi elementi nella nostra applicazione web.

Per poter aggiungere Bootstrap nel progetto di Angular basta seguire i seguenti passi.

1. Installare Bootstrap

Lanciare il seguente comando nel terminale, all’interno della cartella del progetto.

npm install bootstrap

2. Aggiungere i file di Bootstrap

Aprire il file angular.json e aggiungere all’array chiamato styles le seguenti righe.

"node_modules/bootstrap/dist/css/bootstrap.min.css",

"node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"

Ora potete usare Bootstrap nel vostro progetto Angular.
