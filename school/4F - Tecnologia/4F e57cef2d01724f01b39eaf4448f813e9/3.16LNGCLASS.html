NGCLASS

Con la direttiva NGCLASS è possibile specificare dinamicamente la classe CSS di un elemento HTML.

Esempio:

<ul>

 <li *ngFor="let persona of persone; index as i">

   <div class="cerchio" [ngClass]="{'cerchioOnline prova': persona.isOnline, 'cerchio0ffline': persona.isOnline}">

     <p>{{persona.nome}} {{persona.cognome}}</p>

   </div>

 </li>

</ul>

Nell’esempio scritto sopra noi andiamo ad aggiungere al div delle classi CSS in base a determinate condizioni, usando l’NGCLASS.

L’NGCLASS si specifica nel seguente modo: [ngClass]="{}"

All’interno delle parentesi graffe si mette una lista di elementi, separati da virgole.

Ogni elemento è del tipo 'class_list': condition, dove class_list è una lista di classi CSS separate da uno spazio, mentre condition è un qualsiasi tipo di condizione booleana.
