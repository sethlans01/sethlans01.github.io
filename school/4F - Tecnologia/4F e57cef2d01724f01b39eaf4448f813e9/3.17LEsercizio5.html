Esercizio 5 - Lista Contatti

Realizza una web app Angular come segue.

  1. La web app deve avere 1 pagina.
  2. Il tema della web app è creare una lista di contatti.
  3. Crea un componente <lista-contatti> e al suo interno usa il costrutto NGFOR per mostrare tutta la lista dei contatti contenuta nel file Typescript.
  4. Ogni contatto deve essere mostrato a schermo di modo tale che si vedano le seguenti informazioni:
  1. Immagine profilo
  2. Nome e cognome
  3. Numero di telefono
  1. Inoltre, ogni contatto ha associato a sé un pallino colorato:
  1. Se il contatto è online, il pallino è verde
  2. Se il contatto è offline, il pallino è rosso
  3. Usa il costrutto NGIF per realizzare questa funzionalità
  1. Realizza nel file typescript una struttura dati che permetta di contenere le seguenti informazioni di un contatto:
  1. Nome
  2. Cognome
  3. Età
  4. Numero di telefono
  5. isOnline (valore booleano)
  6. Colore preferito
  1. Usare la direttiva NGSTYLE per impostare lo sfondo della card / div di ogni contatto
  1. Usa dei colori che non siano un pugno in un occhio pls
  1. Dare ai componenti una struttura ad albero sensata
  2. Il progetto deve essere committato e pushato su una repo di GitHub

NB: QUESTO PROGETTO VERRÀ VALUTATO: chiunque non si metta a fare il progetto o non lo consegni completo entro le scadenze previste, prenderà una nota disciplinare e un voto in meno. (Alla fine di ogni lezione controllo quello che avete fatto).
