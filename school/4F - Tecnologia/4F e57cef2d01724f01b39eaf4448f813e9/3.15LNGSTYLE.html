NGSTYLE

La prossima direttiva che vediamo è NGSTYLE e permette di specificare le proprietà CSS di un tag HTML di un componente in modo dinamico.

Esempio:

<ul>

 <li *ngFor="let persona of persone">

   <div class="cerchio" [ngStyle]="{

     'background': persona.isOnline ? 'green' : 'red',

     'text-align' : 'center',

     'border-radius': 5,

     'color': textColor()

 }">

     <p>{{persona.nome}} {{persona.cognome}}</p>

   </div>

 </li>

</ul>

Per poter specificare le proprietà CSS di un elemento HTML in modo dinamico occorre aggiungere al suo tag HTML il seguente costrutto:

[ngStyle]="{}"

Dove all’interno delle parentesi graffe c’è una lista separata da virgole di proprietà CSS e dei loro valori.

Nel codice scritto sopra sono riassunti tutti i modi con i quali si può specificare una proprietà CSS con questo costrutto:

  • In ogni modo si specifica nella parte sinistra il nome della proprietà all’interno di due apostrofi e dopo il simbolo : si specifica il valore.
  • Il valore può essere un valore semplice come un numero o una stringa (text-align e border-radius)
  • Il valore può essere ottenuto con il ternary operator (background)
  • Il valore può essere ottenuto come risultato di una funzione specificata nel file typescript del componente di angular (color)
